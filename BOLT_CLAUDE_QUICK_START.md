## ðŸ”§ Troubleshooting

### "Can't copy HTML from Bolt"

**Solution:**
```
1. Right-click on page â†’ View Page Source
2. Or press Ctrl+U (Windows) / Cmd+U (Mac)
3. Select all (Ctrl+A / Cmd+A)
4. Copy (Ctrl+C / Cmd+C)
```

### "Images aren't showing"

**Solution:**
```
1. Don't worry! This is normal
2. Go to each image block after import
3. Click "Upload" and add your images
4. Or use "Search Stock Photos"
5. Or paste new image URL
```

### "Some sections missing"

**Solution:**
```
1. Import basic structure first
2. Manually add missing sections
3. Or import that section separately
4. Or use platform's block library
```

### "Colors look wrong"

**Solution:**
```
1. Use AI Color Palette tool
2. Input your brand description
3. Apply generated theme
4. Or manually adjust in settings
```

## ðŸ’¬ Common Questions

**Q: Will my Bolt animations work?**
A: Simple transitions yes, complex JavaScript no. Use platform's animation tools.

**Q: Can I import multiple pages?**
A: Yes! Import each page separately, or import one and duplicate.

**Q: What about forms?**
A: Form structure imports, then connect to your email service.

**Q: Do I need to credit Bolt/Claude?**
A: No, the HTML output is yours to use.

**Q: Can I import from Webflow/Wix?**
A: Use "Export HTML" from those platforms, then import here.

**Q: Will images be hosted?**
A: External images link to original. Upload your own for full control.
