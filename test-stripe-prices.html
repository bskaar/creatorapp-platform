Stripe Price Validation

Checking Stripe prices...

If prices don't exist, create them:

  1. Go to Stripe Products
  2. Create a product called "Growth Plan" with price $99/month recurring
  3. Create a product called "Pro Plan" with price $199/month recurring
  4. Copy the Price IDs (they start with "price_")
  5. Update the subscription_plans table with these Price IDs
